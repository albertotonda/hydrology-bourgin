Une nouvelle version de la base de données CAMELS-FR est en train d'être finalisée au sein de l'équipe.
Elle sera diffusée sur l'entrepôt INRAE https://entrepot.recherche.data.gouv.fr/dataverse/inrae très prochainement.
Le fichier GR2M__4ML.csv contient l'ensemble des données nécessaires pour réaliser une modélisation pluie-débit au pas de temps mensuel sur plus de 500 bassins français.
Ces données pourront être actualisées d'ici le mois de mars.
